Asteroids Smasher
=================

.. image:: screenshot.png
   :width: 50%

An asteroids clone made with the Arcade library.

Designed to show off using shaders for visual effects.

   :widths: 25 75
   :header-rows: 1

     - Bootstrap method that loads the fonts and starts the game.
   * - asteroid_sprite.py
